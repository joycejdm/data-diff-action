# Joyce's Data Diff Action (Em Construção!)

Este é um projeto #buildinpublic para construir uma alternativa de "data diff" para a comunidade dbt-core.

## A Missão
A funcionalidade de "data diff" (que valida o impacto de um PR nos seus dados *antes* do merge) é incrível, mas hoje está muito focada em planos Enterprise caros.

O objetivo deste repositório é criar uma GitHub Action simples, poderosa e acessível para que *toda* a comunidade dbt-core (que usa GitHub Actions, Airflow, etc.) possa ter essa segurança no seu CI/CD.
